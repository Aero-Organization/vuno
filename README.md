# Vuno

A lightweight, cross-platform terminal text editor written in pure Python.

![Alt text](public/banner.png)

## How to run
```bash
vuno [filename]

# Development script run command
python run.py [filename]
```

## Keyboard Shortcuts

### File Operations
| Shortcut | Action |
|----------|--------|
| `Ctrl+X` | Exit editor |
| `Ctrl+O` | Save file |
| `Ctrl+S` | Save as (new filename) |

### Edit Operations
| Shortcut | Action |
|----------|--------|
| `Ctrl+Z` | Undo last action |
| `Ctrl+Y` | Redo last undo |
| `Ctrl+K` | Cut current line |
| `Ctrl+C` | Copy current line |
| `Ctrl+U` | Paste clipboard |

### Search
| Shortcut | Action |
|----------|--------|
| `Ctrl+W` | Search for text |
| `Ctrl+N` | Find next occurrence |

### Navigation
| Shortcut | Action |
|----------|--------|
| `Ctrl+T` | Go to line number |
| `Arrow Keys` | Move cursor |
| `Home` | Go to line start |
| `End` | Go to line end |
| `Page Up/Down` | Scroll page |

### View
| Shortcut | Action |
|----------|--------|
| `Ctrl+L` | Toggle line numbers |
| `Ctrl+I` | Show statistics |
| `Ctrl+G` | Show help |
